This line graph showcases the correlation between two key variables - the half angle (in degrees) and the speed (in km/s) of CMEs - over a specific time frame. Each dot on the chart represents a single CME event, with the x-axis denoting the half angle and the y-axis representing the speed. By connecting these data points, the chart reveals important trends and patterns in CME behavior over time. Furthermore, the chart is equipped with tooltips that provide detailed information about each data point, such as the speed and half angle, making it easier for users to interact with and understand the data. To access this data, the code utilizes NASA's API, specifically the CMEAnalysis endpoint, with the help of an API key for secure authentication. Ultimately, this graph aims to offer valuable insights into the characteristics of CME events within the specified time period.
